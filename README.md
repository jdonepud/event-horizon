# Event Horizon: Automated Vulnerability Triage & Reachability Engine\n\nEvent Horizon is a high-performance security tool designed to automate the ingestion, scoring, and analysis of vulnerabilities. It bridges the gap between raw vulnerability reports and actionable risk intelligence by implementation of CVSS 3.1 standards and advanced vector chaining analysis.\n\n## Key Features\n- **Automated CVSS Scoring**: Precision calculation of Base and Environmental scores using the First.org CVSS 3.1 specification.\n- **Vector Chaining Analysis**: Identifies complex attack paths where multiple vulnerabilities combine to create high-impact reachability.\n- **High-Fidelity Dashboard**: A glassmorphic, real-time visualization layer for monitoring ingestion pipelines.\n- **RESTful API**: Built with FastAPI for seamless integration.\n\n## Project Structure\n- `backend/`: Core logic and API implementation.\n- `frontend/`: Real-time dashboard interface.\n- `automate_triage.py`: CLI tool for rapid processing.\n- `sample_vulns.csv`: Example dataset.\n\n## Getting Started\n### Prerequisites\n- Python 3.8+\n- FastAPI & Uvicorn\n\n### Installation\n1. Clone the repository:\n```bash\ngit clone https://github.com/jdonepud/event-horizon.git\ncd event-horizon\n```\n2. Install dependencies:\n```bash\npip install -r requirements.txt\n```\n3. Run the API:\n```bash\npython backend/main.py\n```
